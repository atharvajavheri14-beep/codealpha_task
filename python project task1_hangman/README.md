# CodeAlpha Python Internship Projects

This repository contains Python programming projects completed as part of the CodeAlpha internship program.

## Projects

### Task 1: Hangman Game
A text-based Hangman game where the player guesses a hidden word one letter at a time.

**Run:**
```bash
python task1_hangman.py
```

**Features:**
- 5 predefined words chosen at random
- ASCII art hangman that updates with each wrong guess
- 6 incorrect guesses allowed
- Tracks all guessed letters

---

### Task 2: Stock Portfolio Tracker
A CLI-based stock tracker that calculates total portfolio investment value.

**Run:**
```bash
python task2_stock_tracker.py
```

**Features:**
- 8 hardcoded stock prices (AAPL, TSLA, GOOGL, MSFT, AMZN, META, NFLX, NVDA)
- User inputs stock symbol and quantity
- Displays formatted portfolio summary table
- Option to save results to a CSV file

---

### Task 4: Basic Chatbot (PyBot)
A rule-based chatbot that responds to common conversational inputs.

**Run:**
```bash
python task4_chatbot.py
```

**Features:**
- Responds to greetings, farewells, questions
- Can tell jokes and the current time
- Handles unrecognized inputs gracefully

---

## Requirements

No external libraries required — all projects use Python standard library only.

**Python version:** 3.7+
